#### http 통신

- 통신방법 (요청 방법)

  - Get

    : 데이터 get (select)

  - Post

    : 데이터 추가 (insert)

  - Put

    : 데이터 수정 (update)

  - Delete

    : 데이터 삭제 (delete)





#### stateless

: 요청마다 스트림 연결해서 data 주고받는 방식

<-> stateful (연결 지속 방식)





#### MIME 타입

: 데이터 타입 (클라이언트에게 전송된 문서의 다양성)

ex) text/html , image/jpeg , video/mp4 , application/js

- 헤더

  : data 설명 (클라이언트 정보, 순서 등)

- 바디

  : data
